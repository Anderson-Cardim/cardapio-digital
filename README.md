# Cardápio Digital 

## Sobre o Projeto

Sistema web que digitaliza cardápios de restaurantes, permitindo que clientes visualizem o menu através de QR Code e façam pedidos direto pelo WhatsApp. Proprietários gerenciam tudo através de um painel administrativo intuitivo.

## Tecnologias Utilizadas

- **React** - Estrutura semântica
- **TaiwindCSS** - Estilização moderna (Gradientes, Flexbox, Grid, Animações)
- **JavaScript (ES6+)** - Lógica e interatividade
